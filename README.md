## 🌐 Research Focus: RAQT Framework
The **Robust Anonymous Quantum Transmission (RAQT)** framework is my primary research project. It aims to optimize 2-node quantum protocols over a simulated 30km relay, focusing on:

* **Noise Mitigation:** Handling thermal noise and decoherence in remote environments.
* **Hardware Constraints:** Optimizing for semiconductor physical properties like effective mass mismatch.
* **Security:** Implementing Post-Quantum Cryptography (PQC) and "Blue Team" cybersecurity operations.

## 🛠️ Global Setup & Requirements
To run the simulations and circuits in this repository, you must have a functional **NetSquid**, **Qiskit**, and **Cirq** environment.

1.  **Prerequisites:**
    * Python 3.8 - 3.11
    * Registered account at [NetSquid.org](https://netsquid.org) for networking libraries.

2.  **Installation:**
    ```bash
    # Create a virtual environment
    python3 -m venv q_env
    source q_env/bin/activate

    # Install Multi-Stack SDKs (Qiskit & Cirq)
    pip install qiskit qiskit-aer cirq matplotlib pylatexenc

    # Install NetSquid (Replace with your credentials)
    pip install --extra-index-url https://{username}:{password}@pypi.netsquid.org netsquid
    ```
